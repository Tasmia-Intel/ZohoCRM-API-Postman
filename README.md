# ZohoCRM-API-Postman

##  Project Overview
This project demonstrates end-to-end API testing of **Zoho CRM APIs** using **Postman** and **Newman**.
It covers OAuth 2.0 authentication, CRUD operations on Leads, negative test cases, schema validation,
data-driven testing, and CLI execution.


---

##  Tools & Technologies
- Postman
- Newman (CLI)
- Zoho CRM REST APIs
- OAuth 2.0
- JavaScript (Postman Scripts)
- Git & GitHub

---

## Features Covered
- OAuth 2.0 token handling (Access & Refresh token)
- Conditional token generation
- Create, Get, Update Lead APIs
- Positive and Negative test cases
- Schema validation using Postman tests
- Data-driven testing using JSON data
- Collection execution using Newman CLI
- HTML report generation
- Secure handling of secrets using `.gitignore`
